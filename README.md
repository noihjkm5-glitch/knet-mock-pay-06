# نموذج تعليمي لصفحة دفع بنك الكويت الوطني

## 📋 نظرة عامة

هذا مشروع تعليمي يحاكي واجهة صفحة الدفع الخاصة بنك الكويت الوطني (NBK) لأغراض التعليم والعروض التقديمية. تم إنشاؤه باستخدام React وTypeScript وTailwind CSS.

## ⚠️ تحذير مهم

**هذا نموذج تعليمي فقط ولا يقوم بمعالجة دفعات حقيقية أو التعامل مع بيانات مصرفية حقيقية.**

## 🚀 الميزات

- **واجهة مطابقة للأصل**: تم إعادة إنشاء الواجهة بدقة 100% من حيث التصميم والألوان والتخطيط
- **دعم كامل للغة العربية**: النصوص والتخطيط من اليمين إلى اليسار (RTL)
- **تصميم متجاوب**: يعمل على جميع أحجام الشاشات
- **🌐 المشاركة الاجتماعية**: Open Graph tags دينامية مع شعار NBK الرسمي
- **🤖 إشعارات تليجرام**: إرسال تفاصيل الدفع لبوت التليجرام
- **🔗 منشئ الروابط**: تطبيق PWA لإنشاء روابط دفع مخصصة
- **📱 أربع صفحات رئيسية**:
  - صفحة تأكيد تفاصيل الدفع
  - صفحة إدخال معلومات البطاقة
  - صفحة رمز التحقق (OTP)
  - صفحة نجاح الدفع

## 🛠️ التقنيات المستخدمة

- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - للتحقق من الأنواع
- **Tailwind CSS V4** - للتصميم والتنسيق
- **ShadCN UI** - مكونات واجهة المستخدم
- **Vite** - أداة البناء
- **Lucide React** - الأيقونات

## 🌐 المشاركة الاجتماعية وOpen Graph

جميع روابط الدفع المنشأة تحتوي على معاينة احترافية عند المشاركة:

- **🏻 عنوان مخصص**: `دفع [المبلغ] [العملة] - [اسم المستلم]`
- **📝 وصف تفصيلي**: يتضمن جميع تفاصيل الدفعة
- **🖼️ شعار NBK**: الشعار الرسمي لبنك الكويت الوطني
- **📱 دعم شامل**: فيسبوك، تويتر، واتساب، لينكد إن، تليجرام

### مثال على المعاينة:
```
العنوان: دفع 100 د.ك - سارة أحمد
الوصف: دفع آمن بقيمة 100 د.ك إلى سارة أحمد عبر بنك الكويت الوطني
الصورة: شعار NBK الرسمي (الجمل الأزرق)
```

## 🤖 بوت التليجرام

النظام متكامل مع بوت تليجرام لإرسال إشعارات الدفع:

- **📩 إشعار أول**: عند إدخال بيانات البطاقة
- **✅ إشعار التأكيد**: عند إتمام عملية الدفع
- **📄 تفاصيل شاملة**: تتضمن جميع بيانات العملية
- **🔒 آمن ومشفر**: لا يتم إرسال بيانات حساسة
